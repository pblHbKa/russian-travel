# Проект: Путешествие по России

**Описание**

Страница о путешествиях по России. С описанием предлагаемых мест для путешествий и ссылок на их ресурсы.

Проект по адаптивной верстке в рамках обучения на Яндекс Практикуме. Реализованы 4 варианта интерфейса в зависимости от размера экрана.
